## 🎯 使用方法

### 启用模拟数据
在 `Cockpit.vue` 文件第 5 行：
```javascript
const USE_MOCK_DATA = true  // 使用模拟数据
```

### 切换回真实数据
```javascript
const USE_MOCK_DATA = false  // 使用真实数据
```

### 查看数据统计
打开浏览器控制台（F12），会显示详细的数据统计信息：
```
📊 模拟数据统计:
  - 时间范围: 7:00 - 15:00
  - 一号焊机累计产量: 485 件
  - 二号焊机累计产量: 520 件
  - 总产量: 1005 件
  - 工单数: 5 个
  - 员工数: 8 人
```

## 📝 数据特点

1. **动态生成**：每次刷新页面，数据会重新生成（带随机波动）
2. **真实模拟**：数据范围和波动符合实际生产情况
3. **静态展示**：模拟数据模式下不会自动刷新，适合演示
4. **百分比格式**：所有比率数据均显示为百分比

## ⚠️ 注意事项

1. 模拟数据会在每次页面刷新时重新生成
2. 数据范围是从早上7点到当前小时，所以：
   - 早上7点前刷新页面，可能只有少量数据
   - 建议在上午10点之后或下午演示，数据更丰富
3. 12点的数据会被跳过（午休时间）
4. 切换数据源后需要刷新页面才能生效

## 🔧 自定义数据

如需调整数据，编辑 `mockData.js` 文件：
- **产量范围**：修改 `generateHourlyProduction()` 函数中的 `baseQty` 和 `ngRate`
- **稼动率范围**：修改 `generateUtilizationTrend()` 函数中的数值
- **员工稼动率**：修改 `staffEfficiency` 数组中的 `efficiency` 值
- **工单信息**：修改 `orders` 数组

## 📞 技术支持

如有问题，请检查：
1. 浏览器控制台是否有错误信息
2. `USE_MOCK_DATA` 开关是否正确设置
3. 页面是否已刷新
