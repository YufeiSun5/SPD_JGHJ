# 安装依赖

## 📦 安装 MySQL 驱动

项目需要使用 MySQL 驱动,请运行:

```bash
go get -u gorm.io/driver/mysql
```

## 📋 完整依赖列表

```bash
# 已安装的依赖
go get -u github.com/gin-gonic/gin
go get -u github.com/eclipse/paho.mqtt.golang
go get -u gorm.io/gorm

# 需要新增的依赖
go get -u gorm.io/driver/mysql
```

## 🔧 更新 go.mod

运行以下命令更新依赖:

```bash
go mod tidy
```

## ✅ 验证安装

运行项目:

```bash
go run main.go
```

如果看到以下输出,说明配置成功:

