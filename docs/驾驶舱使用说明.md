# 驾驶舱页面使用说明

## 页面概述

驾驶舱页面（Cockpit.vue）是一个全屏数据可视化大屏，专为"大连斯顿德 · 激光焊接智能驾驶舱"设计。

## 页面布局

### 顶部区域
- **标题栏**：显示"大连斯顿德 · 激光焊接智能驾驶舱"
- **工位信息**：显示"双工位 · 一号机 ② 二号机"
- **数据卡片**（6个）：
  - 累适提体运出率：89%
  - 今日产量：155 件
  - 良品率：99.1%
  - 在制品：12（三班制）
  - 报警数：0
  - 人均效率：86%

### 主内容区域（三栏布局）

#### 左栏
1. **计划总览**：显示生产计划表格
2. **今日产量**：小时精度的柱状图（一机、二机对比）
3. **质量分布**：两个环形图（一号机92%、二号机88%）

#### 中栏（实时监控）
1. **监控视频 1**：一号机 · 工位 A
   - 状态：运行中
   - 参数：焊接功率 3.2kW，焊缝温度 420°C
   
2. **监控视频 2**：二号机 · 工位 B
   - 状态：待机
   - 参数：保养时长 12m

#### 右栏
1. **员工概效**：柱状图显示8个员工的效率
2. **辅助率趋势**：折线图显示一号机和二号机的辅助率趋势

## 添加视频的方法

### 方法1：本地视频文件

1. 将视频文件放置在项目的静态资源目录中：
   ```
   desktop/frontend/src/assets/videos/
   ```

2. 修改 `Cockpit.vue` 的视频源配置：
   ```javascript
   const videoSources = ref({
     video1: '/src/assets/videos/video1.mp4',
     video2: '/src/assets/videos/video2.mp4'
   })
   ```

### 方法2：网络视频流

如果是网络摄像头或RTSP流，需要：

1. 修改视频源为网络地址：
   ```javascript
   const videoSources = ref({
     video1: 'http://192.168.1.100:8080/video1',
     video2: 'http://192.168.1.100:8080/video2'
   })
   ```

2. 如果是RTSP流，需要通过后端转换为HTTP-FLV或HLS格式

### 方法3：使用占位视频进行测试

在正式接入视频之前，可以使用免费的测试视频：

```javascript
const videoSources = ref({
  video1: 'https://test-videos.co.uk/vids/bigbuckbunny/mp4/h264/360/Big_Buck_Bunny_360_10s_1MB.mp4',
  video2: 'https://test-videos.co.uk/vids/bigbuckbunny/mp4/h264/360/Big_Buck_Bunny_360_10s_1MB.mp4'
})
```

## 访问页面

启动项目后，在导航菜单中点击"驾驶舱"即可访问。

路由地址：`/#/cockpit`

## 样式特点

- 深色科技风格（#0a0e27背景）
- 渐变边框和发光效果
- 响应式图表（使用ECharts）
- 视频占位符，在无视频时显示友好提示
- 自动循环播放、静音

## 自定义修改

### 修改数据源
所有模拟数据都在 `Cockpit.vue` 的 `<script setup>` 部分，可以：
- 修改 `planList` 数组改变计划表格数据
- 修改图表初始化函数中的数据数组

### 修改视频参数
在 `.video-info` 区域修改显示的参数信息：
```vue
<span class="video-params">焊接功率 3.2kW</span>
<span class="video-params">焊缝温度 420°C</span>
```

### 修改配色
主要颜色变量在CSS中：
- 主色调：`#00aaff`（蓝色）
- 强调色：`#00ffaa`（青绿色）
- 警告色：`#ffaa00`（橙色）
- 背景色：`#0a0e27`（深蓝）

## 注意事项

1. **视频格式**：建议使用 MP4 (H.264) 格式，浏览器兼容性最好
2. **视频大小**：注意控制视频文件大小，避免加载过慢
3. **自动播放**：现代浏览器要求视频必须静音才能自动播放
4. **实时流**：如需实时视频流，建议使用专业的流媒体服务器
5. **性能**：大屏页面包含多个图表和视频，注意性能优化
