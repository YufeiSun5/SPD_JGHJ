## 🛑 杀掉进程
```powershell

Get-Process | Where-Object { $_.ProcessName -match "wails|node|iot-monitor" } | Stop-Process -Force
```

## 💡 提示

1. **开发时**：使用 `wails dev` 可以热重载，但需要先打包前端
2. **生产打包**：使用 `wails build` 生成独立可执行文件
3. **前端修改后**：必须重新 `npm run build` 然后重启 `wails dev`
4. **Go 代码修改**：`wails dev` 会自动重新编译

## 🚨 常见问题

### Q: 前端修改没生效？
A: 先执行 `npm run build`，再重启 `wails dev`

### Q: 找不到路径？
A: 使用绝对路径：`cd E:\DEV\SPD_JGHJ\desktop`

### Q: 端口被占用？
A: 先杀掉进程：`Get-Process | Where-Object { $_.ProcessName -match "wails" } | Stop-Process -Force`
